Lab 2 Reflection

In Lab 2, I learned about the basics of HTML, including the basic structure of HTML, and I was able to apply this to create the foundation of my web page. Assisting this were tags, which were helpful tools that defined the layout and organization within my website. I was unsure of what most of these even meant at the start, but they are what make up HTML as a markup language. The validator also helped me correct some of the missed tag information that would've caused error when I was coding, which made me realize how important communicating correctly is when working with these languages. The course readings and videos in this module helped me undertsand a lot of these element and semantic rules and relationships, however. Now, I can confidently write a simple web page, being able to tell the web page what I want it to do.

Lab 3 Reflection

Expanding on the previous lab, Lab 3 taught me how to add styles to the document to make it look like the aesthetic websites I see every day on the internet. This is possible thanks to the CSS language that accompanies HTML code, as I was able to apply the basic CSS style structure using internal and external methods that serve to reduce the clutter in my HTML. After I understood the structure, I could start designing parts of the website using selectors, properties, and values. These were relatively simple and allowed me to assign various fonts, backgrounds, colors, and much more to the plain HTML text. However, I realized that it is the specifics that make CSS so useful, being able to modify everything very precisely as long as you can follow its structural rules properly. All in all, on top of writing a web page, I can now tell it how I want it to look.
