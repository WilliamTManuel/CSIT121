Description

  • In this lab, I created responsive web designs that adapted to different screen sizes.
  • I implemented a single-column layout for mobile devices and a multi-column layout for desktops using flexbox, grid, and media queries.
  • Additionally, I allowed users to go to specific parts of the page through internal navigation using link anchors.
  • To improve my code's readability, I used comments and applied consistent measurements across the CSS.
  • Ultimately, using various CSS layout techniques ensured my site looked good on both mobile and desktop screens.
