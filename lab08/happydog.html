Image 2: JPEG Picture of a Happy Dog

Media Description

This is an picture of a smiling, joyful looking dog. They appear to be having a fun photoshoot in an urban area.

File Type Information

This type of file is called a JPEG or JPG (Joint Photographic Experts Group) and is expressed as .jpeg or .jpg at the end of files.
This is one of the most common types of files for images,
and it works the best for photographs due to its high quality and extremely wide range of colors.
It is also a raster file format made of tiny pixels that can create high resolutions.

Why I Chose This Image

I chose this image because I thought the dog looked very happy and positive, so by sharing it,
I hoped others could feel same as this good boy as well; I try to strive for this happiness in my everyday life.
Also, if I had a dog it would probably be of a similar breed, golden retrievers just look so cuddly.
